# CRM Application for Banquet Hall Booking

## 📌 Project Description
This project is a Customer Relationship Management (CRM) system designed for managing banquet hall bookings.

## 🚀 Features
- User Registration & Login
- Banquet Hall Booking
- Customer Data Management
- Booking History
- Admin Dashboard

## 🛠 Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Database: MySQL

## ▶️ How to Run
1. Install dependencies
2. Run server using:
   python app.py
3. Open browser:
   http://127.0.0.1:5000
